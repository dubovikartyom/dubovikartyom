Сегодня мы разобрали основные команды консоли GIT Bash
Команда Значение
git init создать папку GIT в проэкте
git add добавить файл в проэкт GIT
cd указать путь
pwd показать в какой папке нахожусь
git commit закомитить последнее изменение
git log показать историю изменений
git diff референс commit-a показать изменения в коммите
git status показать статус проэкта
git checkout перейти в другую ветку проэкта
git checkout перейти в другую ветку проэкта
git merge соединить версии проэкта
git remoute привязать проэкт к репозиторию
git push добавить изменения в репозиторий
git branch отобразить ветки проэкта
git clone скопировать проэкт с репозитория
git fetch собирает все commit-ы из ветки репозитория
git pull сливает новые commit-ы в ветку, в которой работаете
git merge abord откат до предыдущего commit-а
git reset id commit возврат к конкретному commit-у
git reset id commit --hard возврат + откат к состоянию
git reset HEAD возврат к последнему commit
git revert возврат к commit с его перезаписанием в новый
