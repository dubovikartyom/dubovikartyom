Gulp

  1. Установка Gulp проводится м помощью команды в командной строке: -g gilp -cli
  2. Необходимо перейти в папку с проектом, используя команду cd
  3. И установить Gulp в свой проект, с помощью команды: gnpm init (имена папок проекта должны состоять исключительно из латинских символов)
  4. Установить пакет зависимостей, находясь в корневой папке проекта использовать команду: npm i gulp --save -dev
  5. Создать файл .gitignore и добавить в него папки node_modules, package-lock,json
  6. Создать файл с конфигурациями в корне проекта (gulpfile.js)
  7. В данном файле поднятуть Gulp к проекту, с помощью условия: var gulp = require("gulp");
  8. Синтаксис создания задачи в Gulp: gulp.task("hello",()=>{})
  9. Запуск задачи в Gulp, с помощью команды: gulp "имя задачи"
  10. Если назвать задачу "default", то она будет выполнена сразу при запуске Gulp
  11. Можно запустить сразу несколько задач поочередно: gulp.task('default', gulp.series("hello1", "hello2))
  12. Параллельный запуск задач: gulp.task('default', gulp.parallel("hello1", "hello2))
  13. Копирование файлов: gulp.task('copy-html', ()=>{return gulp.имя_корневой_папки("имя корневой папки/**/*.html") .pope(gulp.dest("dest/."))})
