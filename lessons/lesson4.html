Сегодня было мое четвертое занятие, мы изучили типы элементов страницы:

Тип элемента Свойства
Блочные элементы имеют размер, соответствуют блочной модели, резервируют всю ширину родителя
Строчные элементы не имеют размеров, не соответствуют блочной модели, занимают необходимое место под свое содержимое
Строчно-блочные элементы имеют размер, соответствуют блочной модели, не резервируют ширину родителя

Разобрали новые теги HTML, для правильной разметки страницы по частям, такие как:

Тег Что означает
header шапка сайта
nav меню с ссылками на другие страницы сайта
main основной контент страницы
footer подвал страницы
