Сегодня мы начали изучать CSS. Рассмотрели возможность подключения стилей отдельным файлом, с помощью указания пути к файлу в теге link, внутри header. А так же, два самыъ главных тега, в которых прописуются селекторы CSS, это: div - для блочных элементов и span для строчных. Вложенность селекторов указывается следующим образом:

Метод Синтаксис Описание
ИЛИ selector, selector выбирает все совпадающие селекторы
Прямое вложение selector > selecrot выбирает только те элементы, которые являются дочерними непосредственно по отношению к указанному элементу.
Любая вложенность selector selector выбирает элементы, которые находятся внутри указанного элемента (вне зависимости от уровня вложенности).
Следующий элемент select + select выбирает элемент, который находится непосредственно после указанного элемента.
По атрибуту [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value] этот селектор выбирает все элементы, имеющие данный атрибут или атрибут с определённым значением.

Также рассмотрели новые CSS свойства, такие как:

Свойство Значение
line-height устанавливает интерлиньяж (межстрочный интервал) текста, отсчет ведется от базовой линии шрифта.
letter-spacing определяет интервал между символами
word-spacing устанавливает интервал между словами
text-shadow устанавливает тень текста
font-family устанавливает стиль шрифта
padding внешний отступ элемента
margin внутренний отступ элемента
text-decoration добавляет оформление текста в виде его подчеркивания, перечеркивания, линии над текстом и мигания

И научились оформлять фон, его цвет и добавлять в него изображения:

Свойство Значение
background-color задает цвет фона
background-image задает изображение фона
background-size задает размер фона (height, width, cover, cantain)
background-position положение фона
background-repeat дублирования изображения фона
background-attachment фиксация фона, при перемещении вниз/вверх, по странице
